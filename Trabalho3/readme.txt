Para executar o programa basta executar:

python3 trabalho3.py

Os argumetos que podem ser usados de entrada são:

-r 1,2 ou 3 (qual dos requisitos deseja rodar)
-i baby ou aloe (rodar o requisito 1 com quais imagens)
-w {numero impar} (tamanho da janela de busca do SAD)

O script get_images_stereo.py foi o utilizado para adquirir minhas próprias imagens, mas só funciona caso você tenha 2 webcams conectadas.

Para adquirir os mesmos resultados que os que indiquei no relatório, entre no meu github: https://github.com/TiagoGallo/PVC/tree/master/Trabalho3
e faca o download de todas as pastas de imagens, a execucao do programa espera que esses diretorios de imagens existam e tenham as iimagens corretas neles, portanto sem isso o programa
nao vai fucnionar corretamente
