Para executar qualquer um dos programas basta rodar:

python3 RequisitoX.py  

substituindo o X pelo número do requisito a ser executado.

Os requisitos 1 e 2 estão carregando a imagem "cores.jpg" por default, para testar com outra imagem
execute o programa com:

python3 Requisito1.py --image "<path to image>"

Já o requisito 3 carrega o vídeo cargoPickupGantryTest1.avi por default, para testar com outro video
execute o programa com:

python3 Requisito3.py --video "<path to video>"
